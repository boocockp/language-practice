# Helper syntax improvements

## Overview

Make template helpers easier to use, eg:

- constants for article and tense names
- make word objects act as helpers
- properties of word objects
- compulsory arguments eg tense, subject can occur in any order before options

## Notes

- Can have a helper and a data object with the same name
- OR can use a data object as a function if it has a call method - see asyncHandlebarsTest.js
